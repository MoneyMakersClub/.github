# 📕취향 기반 독서 기록 서비스, BookDuck

북덕(BookDuck)은 독서를 즐기는 사람들을 위한 취향 기반 독서 기록 서비스입니다.
캐릭터 커스텀, 취향 분석 독서 리포트, 나만의 서재 만들기 기능으로 독서 경험을 즐겁고 편리하게 기록할 수 있습니다.
